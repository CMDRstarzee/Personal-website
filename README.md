# 🎮 NES-Style Personal Side Project

A fun, retro-themed personal landing page built with an 8-bit gaming aesthetic. This project was created as a side experiment to explore the **NES.css** framework and classic pixel-art web design.

---

## 🕹️ Project Overview
This is a "digital character sheet" designed to look like a classic Nintendo interface. Instead of a standard corporate layout, it uses speech balloons, pixelated icons, and retro typography to display info.

### **Tech Stack**
* **Framework:** [NES.css](https://nostalgic-css.github.io/NES.css/) (8-bit CSS Framework)
* **Grid:** [Flexbox Grid](http://flexboxgrid.com/) (Responsive Layout)
* **Typography:** "Press Start 2P" via Google Fonts
* **Icons:** NES.css built-in pixel icons

---

## ✨ Features
* **Animated UI Elements:** Uses classic gaming components like coins, hearts, and stars.
* **Progress Bars:** Visualizes "Knowledge Stats" using colored bars (Success, Warning, and Error levels).
* **Speech Balloons:** A retro greeting system for a personal touch.
* **Social Connectivity:** Pixel-styled social media icons linked to external profiles.
* **Fully Responsive:** Adapts to different screen sizes while maintaining the pixel-perfect look.

---

## 🚀 Quick Start
Since this project is built using strictly **HTML and CSS (CDN-based)**, there is no complex setup required.
  **Run Locally:**
    Open `8bit.html` in your favorite web browser.

---

## 🛠️ Customization
To swap out the stats or info for your own:
* **Stats:** Change the `value` in any `<progress>` tag (e.g., `value="80"`).
* **Themes:** Change the progress bar colors using classes like `is-success` (green), `is-warning` (yellow), or `is-error` (red).
* **Avatars:** The current experience section uses pixelated GitHub avatars.
